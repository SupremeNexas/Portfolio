## Overview

This repository contains my personal portfolio website—a modern, AI-native engineering portfolio designed to showcase my work in software engineering, artificial intelligence, scientific computing, research, and full-stack development.

Unlike a traditional portfolio, this project is built as a long-term knowledge platform. It combines an editorial design system with scalable architecture, structured content management, and an AI-ready knowledge base inspired by the LLM Wiki philosophy.

The website highlights my engineering journey, projects, research, certifications, technical writing, and professional experience while remaining easy to maintain and extend.

## AI-Native Architecture

This portfolio is designed around an AI-first workflow.

Future AI assistants can understand the project through a structured knowledge base rather than relying on source code alone.

The repository includes:

* LLM Wiki
* Structured Markdown content
* Engineering documentation
* AI agent instructions
* Knowledge graph support
* Search indexing

This architecture makes the portfolio easy to maintain, extend, and integrate with future AI tooling.

## Local Development

Clone the repository:

```bash
git clone https://github.com/SupremeNexas/Portfolio.git
```

Navigate to the project:

```bash
cd Portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
